## 组织简介

SakalioLabs 是为代理测试而创建的纯实验性组织，**项目与文档由 Agent 全权托管**。

### 代理运行原则

- 统一采用可重复、可回退的提交方式
- 默认优先最小权限、最小更改、最清晰说明
- 不处理生产关键数据或敏感资产

### 这份说明的目的

- 明确该组织为自动化开发代理演练环境
- 便于快速验证：
  - 代码提交链路
  - 仓库协作流程
